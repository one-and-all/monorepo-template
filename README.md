## 폴더 구조

```
src
 ┣ apps # 각자 개발은 app 폴더에
 ┃ ┣ onasys # 원앤올에서 제공하는 기능 관련 모듈은 onasys 폴더로 제공
 ┃ ┃ ┣ board
 ┃ ┃ ┣ survey
 ┃ ┃ ┣ ...
 ┃ ┃ ┗ package.json
 ┣ packages # 공통 모듈이 위치할 폴더
 ┃ ┣ components
 ┃ ┃ ┣ ...
 ┃ ┃ ┗ package.json
 ┃ ┣ example
 ┃ ┃ ┣ ...
 ┃ ┃ ┗ package.json
```
